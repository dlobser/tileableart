ReadMe!

Code is a powerful expressive medium and the walls that have kept creative people away from it are becoming thinner.
When artists learn to control the algorithm, a new set of styles will emerge, and from that, a new way of seeing.
This site is the beginning of an attempt to lay out a series of skill building demos.
There are a lot of excellent tutorials out there but the idea here is to create a series of sandboxes with links to learn more.
